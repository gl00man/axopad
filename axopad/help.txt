Axopad help:

New > Create a new file
Open > Open existing file
Save > Save the current file
Save As > Save current file as new file to specyfic path
Settings > Set your font family and font size

Shortcuts:

CTRL + N > New file
CTRL + O > Open file
CTRL + S > Save file
CTRL + SHIFT + S > Save file as
CTRL + F > Find phrase
CTFL + H > Settings
CTRL + X > Exit
